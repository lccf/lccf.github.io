### 环境配置

```bash
mkdir ts-cucumber
cd ts-cucumber
mkdir -p features/step_definitions features/support src/step_definitions src/support
echo '{}' > package.json
npm install typescript cucumber webdriverio cucumber-assert @types/cucumber @types/webdriverio --save-dev
```

tsconfig.json

```javascript
{
    "compilerOptions": {
        "module": "commonjs",
        "target": "es6",
        "noImplicitAny": false,
        "sourceMap": true,
        "removeComments": false,
        "lib": [
            "es6",
            "dom",
            "scripthost",
            "es2015.generator",
            "es2015.promise"
        ],
        "sourceRoot": "src",
        "outDir": "features"
    }
}
```

package.json

```javascript
"scripts": {
    "start": "tsc -p . && cucumber"
  },
```

features/bing_search.feature 

```Gherkin
Feature: 必应搜索
  Scenario: 搜索cucumber
    Given 我访问 "https://www.bing.com"
    When 我输入 "cucumber"
    And 我点击 "搜索"
    Then 页面标题中应包含 "cucumber"
```
